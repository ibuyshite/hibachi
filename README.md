# hibachi volume bot
created just for fun - forked from [Yyeger](https://github.com/Yyeger/hibachi-xyz-volume-bot)
#requirements

```apt update```
```apt install software-properties-common -y```
```add-apt-repository ppa:deadsnakes/ppa -y```
```apt update```
```apt install python3.13 python3.13-venv python3.13-dev -y```

then activate environment 
```python3.13 -m venv hibachi-env```
```source hibachi-env/bin/activate```



add an .env file with account_id private_key and api_key
